# Neo Soul Chord Engine for Renoise

Neo Chords 2 is a Renoise tool that generates and writes neo-soul chord progressions directly into instrument phrases. It ships with a baked database of 200 chord progressions you can browse by root key, or recombine into new progressions using the built-in randomizer. From there, you can transpose, humanize timing and velocity, apply a groove pattern, and render the result as simultaneous chords, a strum, or an arpeggio — all without leaving Renoise.

For full usage details, see the [manual](manual.md).
